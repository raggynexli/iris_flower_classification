# House Price Prediction

## Project Overview

This project predicts house prices based on the area of the house and the number of bedrooms using Machine Learning.

## Objective

To build a Linear Regression model that can estimate the price of a house from its features.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Dataset

The dataset contains:

* Area
* Bedrooms
* Price

## Machine Learning Algorithm

Linear Regression

## Project Workflow

1. Load Dataset
2. Select Features and Target
3. Split Data into Training and Testing Sets
4. Train Linear Regression Model
5. Evaluate Model Performance
6. Predict House Prices
7. Visualize Results using Graphs

## Results

* Mean Absolute Error (MAE): 556603.77
* R² Score: 0.9796

## Features

* Predicts house prices based on user input.
* Displays model performance metrics.
* Generates Actual vs Predicted Price graph.

## How to Run

Install required libraries:

pip install pandas matplotlib scikit-learn

Run the program:

python house_price_prediction.py
